Behold my Projects

  • "Idle Head Trauma"

    I never got the point "idle games." What's so fun about a game that plays itself, right? But then I was assigned to create UI assets for a mobile idle game. The more I researched about this genre, the more I wanted to whack my face in with a baseball bat. And that's when it hit me (no pun intended). I should make assets for an idle game where you just repeatedly hit yourself in the face with various items!

  • "Intuitive Investigation"

    In one of my Game design classes, I worked on a game I called "Intuitive Investigation" where you solve a murder using nothing but intuition to figure out the weapon, time, location, motive, and finally the actual murderer. I never really understood how it worked but I made these slick looking cards to go with it, as well as that logo!
